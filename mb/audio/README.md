# Audio Files Setup Guide

## Background Music for Each Parva

Place MP3 files in this directory with the following names:

### Peaceful/Wisdom Parvas (Soft Instrumental)
- `parva-03-vana.mp3` - Forest sounds, flute, peaceful
- `parva-12-shanti.mp3` - Veena, spiritual, calm
- `parva-13-anushasana.mp3` - Meditative, soft tabla
- `parva-15-ashramavasika.mp3` - Serene, hermitage ambience

### Epic/Divine Parvas (Orchestral/Majestic)
- `parva-01-adi.mp3` - Creation theme, divine
- `parva-05-udyoga.mp3` - Diplomatic, building tension
- `parva-14-ashvamedhika.mp3` - Royal ceremony, majestic
- `parva-18-svargarohana.mp3` - Celestial, ethereal

### War/Battle Parvas (Drums, Intense)
- `parva-06-bhishma.mp3` - Epic war drums, dramatic (Contains Gita)
- `parva-07-drona.mp3` - Intense battle, fast tempo
- `parva-08-karna.mp3` - Tragic heroic theme
- `parva-09-shalya.mp3` - Final battle climax

### Dramatic/Emotional Parvas
- `parva-02-sabha.mp3` - Tension, ominous building
- `parva-04-virata.mp3` - Mystery, disguise theme
- `parva-10-sauptika.mp3` - Dark, night ambience
- `parva-11-stri.mp3` - Sorrowful, slow, grieving
- `parva-16-mausala.mp3` - Destruction, dark tones
- `parva-17-mahaprasthanika.mp3` - Journey, contemplative

## File Format Requirements
- **Format**: MP3 (most compatible)
- **Bitrate**: 128kbps (good quality, small size)
- **Duration**: 2-5 minutes (will loop)
- **Volume**: Normalized (consistent across all files)

## Quick Setup (No Music Yet)
The website will work without audio files. Music controls will appear but show "Add audio files" message.

## Full Setup
1. Download tracks from sources above
2. Rename them according to the list
3. Place in this `/audio/` directory
4. Reload the website

The music will automatically play when scrolling to each parva!
